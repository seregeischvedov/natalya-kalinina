# Папка для изображений

Эта папка предназначена для хранения всех изображений сайта.

## Необходимые изображения:

### Обязательные:
1. **profile-photo.jpg** — Фото профиля для главной страницы и "Обо мне" (600x800px или выше)
2. **therapy-session.jpg** — Фото для страницы "Долгосрочная терапия"
3. **psychoanalysis.jpg** — Фото для страницы "Психоанализ"
4. **coaching.jpg** — Фото для страницы "Сопровождение руководителей"
5. **profile-full.jpg** — Полное фото для страницы "Обо мне"

### Рекомендуемые форматы:
- Портреты: 600x800px (вертикально)
- Горизонтальные фото: 1200x800px
- Формат: JPG (оптимизированный, <500 KB на файл)

### Требования к фото:
- ✅ Профессиональное качество
- ✅ Хорошее освещение
- ✅ Нейтральный или размытый фон
- ✅ Солидный и экспертный образ
- ✅ Оптимизированы для веб (не более 500 KB)

## Как добавить изображения:

1. Поместите файлы в эту папку (`images/`)
2. Убедитесь, что имена файлов совпадают с указанными выше
3. Проверьте, что изображения загружаются на всех страницах

## Альтернатива:

Если фотографий нет, сайт будет использовать placeholder-изображения (https://placehold.co/...), которые уже настроены в коде.
